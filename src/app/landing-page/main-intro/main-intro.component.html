Knora ui

User Interface Modules for Knora

This is the demo environment for Knora ui modules, which will be published on npm.

The modules helps to create a graphical user interface for Knora in a quick and simple way. They're written in Angular (version 6) including the material design.

Knora itself is a software framework for storing, sharing, and working with primary sources and data in the humanities.

It is developed by the Digital Humanities Lab at the University of Basel, and is supported by the Swiss Academy of Humanities and Social Sciences.
